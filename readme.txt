This plugin rounds off Woocommerce Cart Total to nearest 10 cents (or nearest 10 in the 2nd decimal for any currency).

== Description ==

This plugin rounds off Woocommerce Cart Total to nearest 10 cents (or nearest 10 in the 2nd decimal for any currency). 

For example, if the cart total after tax is 43.12, the plugin rounds it off to 43.20. Similarly if the cart total is 43.28, the plugin rounds it off to 43.30.  

There is no additional setting required to be configured. Simply install the plugin and activiate it. 


== Frequently Asked Questions ==

= What type of rounding is this? =

Rounding is carried out to arrive at the nearest 10 cent or nearest 10 in the 2nd decimal point. This rounding works only for Woocommerce cart total.

= Do I need to install woocommerce for this? =

Yes. this plugin is dependent on woocommerce plugin. 

= How to configure? =

There is no configuration required. Simply install, activate and check the cart total.
